Detailed Description

Given multiple type-lists this algorithm yields a new type-list consisting of the n type-lists (where n is the minimal size of the source type-lists).

Let t0, t1, ..., tn be type-lists with ti[j] denoting the j-th element of the i-th type-list and m the minimum size of the source type-lists. The resulting type-list will then be built like the following pattern:

<
<t0[0], t1[0], ..., tn[0]>,
<t0[1], t1[1], ..., tn[1]>,
...,
<t0[m], t1[m], ..., tn[m]>
>
