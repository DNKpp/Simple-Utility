Detailed Description

template<class T>
requires std::same_as<T, std::remove_cvref_t<T>>
class sl::functional::value_fn< T >

Functional helper type, which takes a value and returns them on invocation.

This type has special treatment for std::reference_wrapper types. Instead of returning those at invocation, they will be unwrapped and the containing reference will be returned instead.

Note
In general the value_type will be equally to T, but if T is an array type, it will become T*. reference_type will be usually equally to const value_type&, but if value_type is a std::reference_wrapper<U>, then it will be U&.
