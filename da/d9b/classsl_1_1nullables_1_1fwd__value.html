Detailed Description

template<class TFunc>
class sl::nullables::fwd_value< TFunc >

Passes the value of the input_nullable to the function if it's not equal to its null-object.

Template Parameters
TFuncThe type of the passed function. The function should be invokable with types returned by value_unchecked.

This algorithm uses the actual value of a input_nullable and passes it to the given functional. If the input_nullable compares equal to its null-object, nothing happens.

Note
In the following examples the outcome is always presented within the REQUIRE() statement.

This example shows what happens when a valid input_nullable is used in a fwd_value expression.

namespace sn = sl::nullables;
int result{ 42 };
std::optional<int> opt{ 1337 };
opt | sn::fwd_value{ [&result](int value){ result = value; } };
REQUIRE(result == 1337);

This example shows what happens when an invalid input_nullable is used in a fwd_value expression.

namespace sn = sl::nullables;
int result{ 42 };
std::optional<int> opt{ std::nullopt };
opt | sn::fwd_value{ [&result](int value){ result = value; } };
REQUIRE(result == 42);
