Definition: adapter.hpp:423

Determines whether a type can be used as a tuple-like.

Requires the type-traits std::tuple_size to be defined for the given type and its member value denoting the the correct tuple size. The std::tuple_element trait and the get function must to be defined for each index in the interval [0, N), where N is the tuple size of the given type. \concept tuple_like
