Detailed Description

Trait determining whether some type-lists have the container type in common.

The container is the actual template of a type-list. For example, given the type-list A<B, C> A denotes the container. Two (or more) type-lists have a common container, if their containers are exactly the same, thus no conversions will be applied. If only one type-list is provided, its container is used.

If no such common container can be determined, the member alias type will be omitted.
