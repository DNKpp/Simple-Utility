Detailed Description

template<class T>
class sl::nullables::value_or< T >

Retrieves the value of a input_nullable if it's not equal to its ''null''-object. Returns the alternative otherwise.

Template Parameters
TType of alternative. Must initialize nullable_value_t<TNullable> objects.

This algorithm returns the alternative if the input_nullable compares equal to its null-object. Otherwise the value of the input_nullable is returned.

Note
In the following examples the outcome is always presented within the REQUIRE() statement.

This example shows what happens when a valid input_nullable is used in a value_or expression.

namespace sn = sl::nullables;
std::optional<int> opt{ 1337 };
const int result = opt | sn::value_or{ 42 };
REQUIRE(result == 1337);
