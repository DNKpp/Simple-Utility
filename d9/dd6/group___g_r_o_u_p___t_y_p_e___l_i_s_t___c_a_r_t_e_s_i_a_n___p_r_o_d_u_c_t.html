Detailed Description

Given multiple source type-lists this algorithm yields a new type-list combining each element of each list with each other.

Let t0, t1, ..., tn be type-lists with ti[j] denoting the j-th element of the i-th type-list and li the size of the i-th source type-lists. The resulting type-list will then be built like the following pattern:

<
<t0[0], t1[0], ..., tn[0]>,
...,
<t0[0], t1[0], ..., tn[ln-1]>,
...,
<t0[0], t1[l1-1], ..., tn[0]>,
...,
<t0[0], t1[l1-1], ..., tn[ln-1]>,
...,
<t0[l0-1], t1[0], ..., tn[0]>,
...,
<t0[l0-1], t1[0], ..., tn[ln-1]>,
...,
<t0[l0-1], t1[l1-1], ..., tn[0]>,
...,
<t0[l0-1], t1[l1-1], ..., tn[ln-1]>
>
