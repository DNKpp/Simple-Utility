Detailed Description

Appends at the end of the first type-list all elements of the other type-lists.

Note
The resulting type-list container will always be the one of the first type-list.

Let t0, t1, ..., tn be type-lists with ti[j] denoting the j-th element of the i-th type-list and li the length of the i-th type-list. The resulting type-list will then be built like the following pattern:

<t0[0], t0[1], ..., t0[l0 - 1], t1[0], t1[1], ..., t1[l1 - 1], tn[0], tn[1], ..., tn[ln - 1]>
