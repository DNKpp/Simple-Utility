Detailed Description

Removes all element duplicates contained by the source type-list.

The algorithm populates the result type-list with elements beginning from the front of the source type-list. If the current element isn't already contained in the result type-list, then it will be appended.

I.e. let <int, float, int> be a source type-list with arbitrary container type. The result type-list will then be <int, float>, thus omitting the second appearance of int.
