Detailed Description

Queries the given type-list for a specific type.

If the query type is contained in the given type-list, the value member contains the index at which the query type appears inside the type-list. If it's not contained, no such member is defined.

Note
The query type may appear multiple times, but the algorithm finds only returns the index of the first one found (the least index).
